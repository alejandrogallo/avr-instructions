SEI - Set Global Interrupt Flag

Description:

Sets the Global Interrupt flag (I) in SREG (status register). The instruction following SEI will be executed before any pending interrupts.

Operation:

(i) I ← 1

Syntax: Operands: Program Counter:

(i) SEI None PC ← PC + 1
