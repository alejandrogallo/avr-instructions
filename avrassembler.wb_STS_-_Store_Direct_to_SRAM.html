STS - Store Direct to SRAM

This 1-word form of the STS instruction is only used on ATtiny10-based devices. Other AVR devices use the STS .

Stores one byte from a register to the SRAM section of the data space.

An 8-bit address in the range 0x40 <= k <= 0xBF must be supplied. The assembler will calculate an 7-bit offset into the SRAM memory from this address, hence this instruction is capable of addressing up to 128 bytes of SRAM. This instruction cannot be used to store I/O addresses, use the OUT instruction for this purpose.

Operation:

(i)(k) <- Rr

Syntax: Operands: Program Counter:

(i)STS k, Rr 16 ≤ r ≤ 31, 0x40 ≤ k ≤ 0xBF PC <- PC + 1
