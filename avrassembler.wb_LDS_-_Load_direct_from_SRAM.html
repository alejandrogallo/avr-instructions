LDS - Load Direct from SRAM

This 1-word form of the LDS instruction is only used on ATtiny10-based devices. Other AVR devices use the LDS .

Description:

Loads one byte from the SRAM section of the data space to a register.

An 8-bit address in the range 0x40 <= k <= 0xBF must be supplied. The assembler will calculate an 7-bit offset into the SRAM memory from this address, hence this instruction is capable of addressing up to 128 bytes of SRAM. This instruction cannot be used to load I/O addresses, use the IN instruction for this purpose.

Operation:

(i)Rd ← (k)
