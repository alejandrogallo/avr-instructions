AVR Assembler AVR Assembler Preprocessor
========================================

\#endif
-------

### <a href="" id="N10B22"></a> Syntax

`       #endif      `

### <a href="" id="N10B29"></a> Description

Terminates a conditional block initiated with an `       #if      ` , `       #ifdef      ` , or `       #ifndef      ` directive.
